# Fitness-Web-Api-Project-JWT
.Net Core 5.0 Web Api Projesi. Projede AccessToken ve RefreshToken kullanarak katmanlı mimari ile SOLID prensiplerine uygun token bazlı bir kimlik doğrulama , yetkilendirme yapısı inşa ettim.
